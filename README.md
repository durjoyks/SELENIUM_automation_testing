

# Evershop Automation with Selenium

This project automates the process of searching, selecting, and purchasing a product on [Evershop Demo](https://demo.evershop.io/) using **Selenium WebDriver** with **Node.js**.

## Demo Video
See the full automation workflow on Evershop in this screen recording: [Watch the Screen Recording](https://drive.google.com/file/d/1yn1V1Y1w3BaZEu29ZBVHqMCOTkCk6U_X/view?usp=drive_link)

## Features
- Searches for a product (`Nike air zoom pegasus 35`).
- Randomly selects product size and color.
- Adds the product to the cart.
- Proceeds through checkout by filling out form details.
- Selects delivery and payment options.
- Places the order and continues shopping automatically.

## Prerequisites
- [Node.js](https://nodejs.org/) installed
- Google Chrome browser installed
- ChromeDriver (compatible with your Chrome version)

## Installation
1. Clone the repository
2. Install dependencies ( npm install selenium-webdriver chai )
3. Run the automation script ( node test.js )
